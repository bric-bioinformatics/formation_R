Course : Introduction to R for data analysis and representation

Goal of this course

During this course, you’ll get an introduction to the R language and its graphical interface named RStudio. You’ll learn how to use the language and its interface through various exercises.

The aim is to enable you to acquire basic programming skills in R, so that you can manipulate data tables, dissect them and extract the relevant information you need.

In this course, you’ll also learn how to make simple but important graphical representations in biology. You’ll see how to make them aesthetically pleasing and easy to read, so that you can use them in your publications.

Course sheets

  • Introduction to the R language (Slides)
  • Discover R and RStudio : installation and presentation
  • Interpret simple code and create variables
  • First data manipulations
  • Gather information simply: merge() function
  • Notion of packages
  • Manipulate dataframes : dplyr package
  • Represent data : ggplot2 package
